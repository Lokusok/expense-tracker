## Приложение для отслеживания расходов

Фичи:

- Сессии
- CRUD над расходами, изменение профиля пользователя
- Сброс пароля по почте
- Свой MVC, DI
- Composer
- Интерактив на клиенте при помощи Alpine
- Контейнеризация при помощи Docker / Docker-Compose
